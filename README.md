## 文件结构：
```
  ├── model.py: ResNet模型搭建
  ├── train.py: 训练脚本

  ├── pth2onnx.py: 模型转换脚本
  ├── pth2onnx_test.py: 转换后模型推理脚本
  └── batch_predict.py: 批量图像预测脚本

```
